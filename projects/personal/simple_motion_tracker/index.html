Simple motion tracker

April 20, 2022

2022   ·   personal  

As part of this university project, I needed to track a fiducial marker, in time, in a large set of slow-motion videos, and output the marker’s position over time. Instead of doing this by hand using a tool such as Tracker, I decided to write a little tool that does it automatically.

The tool is written in Python and uses OpenCV to do the tracking. The tool uses template matching to find the marker in each frame of a video.
